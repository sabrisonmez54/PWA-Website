Mobile Apps and Services Test Engineer

August 2020 - Present

  • At BMW I am a test engineer supporting the (Spider) Test Automation team by translating user requirements into codable tests, coding the tests, running and reporting on the results of said tests, and then taking the next step in configuring the tests within the Spider Test Automation team's test automation suite to fully automate the test scripts and unit tests.
  • For this process we use various technologies by building an integration from Cucumber to TestCafe in order to better communicate with the product team. We take the Gherkin readable tests and run them using Javascript on various browsers. We then use Jenkins to run the tests automatically and report them from the Jenkins script to Xray for Jira.
  • The project that I am assigned with currently is the Charge Forward program, which BMW is currently testing in the Bay Area in collaboration with the local energy provider Pacific Gas & Electric (PG&E). The aim of the project is to use as much solar energy as possible while keeping the public electricity grid stable.
